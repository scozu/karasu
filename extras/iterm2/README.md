# Karasu Colorscheme for iTerm2

A warm, atmospheric terminal colorscheme for iTerm2 combining Material Design dark backgrounds with warm earth tones.

## Installation

### Method 1: Direct Download
```bash
# Download to iTerm2 schemes directory
curl -o ~/Library/Application\ Support/iTerm2/Dynamic\ Profiles/karasu.itermcolors https://raw.githubusercontent.com/scozu/karasu/main/extras/iterm2/karasu.itermcolors
```

### Method 2: Manual Installation
1. Copy `karasu.itermcolors` to your iTerm2 schemes directory:
   ```bash
   cp extras/iterm2/karasu.itermcolors ~/Library/Application\ Support/iTerm2/Dynamic\ Profiles/
   ```

2. Open iTerm2

3. Import the color scheme:
   - Open Preferences (Cmd + ,)
   - Go to Profiles > Colors
   - Click the gear icon at the bottom right
   - Select "Import..."
   - Choose `karasu.itermcolors`

4. Select the "Karasu" profile or set it as default

## Configuration

The theme includes:
- 16 ANSI colors (8 normal + 8 bright)
- Background and foreground colors
- Cursor colors
- Selection colors
- True color support

## Color Palette

- **Background**: #121212 (Material dark)
- **Foreground**: #d4c5b9 (Warm cream)
- **Cursor**: #d8a657 (Bright yellow)
- **Selection**: #2d3437 (Visual selection)

## Customization

You can customize colors after importing:
1. Go to Profiles > Colors
2. Select the Karasu profile
3. Modify any color as needed
4. Save changes to create a custom variant

## Requirements

- iTerm2 terminal emulator
- macOS

## Troubleshooting

If colors don't appear correctly:
1. Ensure iTerm2 is updated to the latest version
2. Check that the .itermcolors file is properly imported
3. Verify the profile is selected and active
4. Restart iTerm2 completely
